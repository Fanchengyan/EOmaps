===============

The following chapters provide comprehensive overview on the capabilities of EOmaps.

|  :doc:`api_basics`
|  Introduction of the basic concepts and how to get started.

|  :doc:`api_data_visualization`
|  Overview on how to visualize datasets (plot-shapes, classification, colorbars etc.).

|  :doc:`api_companion_widget`
|  The graphical user interface that ships with EOmaps!

|  :doc:`api_callbacks`
|  Turn your maps into interactive widgets.

|  :doc:`api_webmaps`
|  Add imagery provided by WebMap services (ts, wms, wmts, xyz) to the map.

|  :doc:`api_naturalearth_features`
|  Add basic map features (coastlines, ocean-coloring etc. ) to the map.

|  :doc:`api_geodataframes`
|  Add ``geopandas.GeoDataFrames`` to the map.

|  :doc:`api_annotations_markers_etc`
|  Customize the map with markers, annotations, lines, logos etc.

|  :doc:`api_scalebar`
|  Add a scalebar to the map.

|  :doc:`api_compass`
|  Add a compass (or North Arrow) to the map.

|  :doc:`api_gridlines`
|  Add grid-lines (and optionally grid-labels) to the map.

|  :doc:`api_utils`
|  A collection of utility widgets (layer-sliders, layer-selectors)

|  :doc:`api_inset_maps`
|  Create zoomed-in views on specific regions of a map.

|  :doc:`api_draw`
|  Interactively draw geometries on a map and export them as shapefiles.

|  :doc:`api_layout_editor`
|  Interactively re-arrange and re-size axes of a figure.

|  :doc:`api_read_data`
|  Read data from NetCDF, GeoTIFF or CSV files.

|  :doc:`api_misc`
|  Some additional functions and properties that might come in handy.
